このアプリケーションについて

説明:

Web上で開発者が質問・回答し合い、情報を共有するためのフォーラム Webアプリケーションです。

機能:

  • ログイン
    • メールアドレス（ユーザーID）とパスワードでログインします。
    既存のユーザーでログインする場合、次のメールアドレスが使用できます（パスワードはすべて "password"）
    yamada@xxx.co.jp
    sato@xxx.co.jp
    jiro@xxx.com
    ihanako@gmail.com
    syoko@yahoo.co.jp
    testuser1@test.com

  • ユーザー登録
    • ユーザー登録には、メールアドレス（ユーザーID）、ユーザー名（表示名）、パスワードを入力します。

  • 質問の検索
    • キーワードはスペースで区切ります。フレーズ検索する場合は、複数ワードをダブルクォーテーション（""）で囲みます。
    例）
    aws ec2
    java "spring boot"

  • 質問の投稿
    • トップページの [質問する] ボタンをクリックすると入力フォームに遷移します。
    コードブロックやリンクなどの Markdown の使用は未サポートです。
    なお、トップページ以外のページからトップページへ遷移するには、左上のロゴ "Developer Forum" をクリックします。

  • 回答の投稿
    • 質問のリンクをクリックして表示された質問詳細ページの一番下にある [回答する] ボタンをクリックすると、入力フォームが現れます。
    コードブロックやリンクなどの Markdown の使用は未サポートです。

  • 質問・回答の編集
    • ログイン中のユーザーが投稿した質問、回答のみ、ユーザー名の下に "編集" リンクが表示されるので、そのリンクをクリックして編集します。

  • ユーザー情報の表示
    • ログイン後、右上に表示されるユーザー名をクリックすると、ユーザー情報のページへ遷移します。
    ユーザー情報のページでは、ユーザー名、メールアドレス、投稿した質問、回答した質問の一覧を見ることができます。

  • ユーザー情報の編集
    • ユーザー情報ページの "編集" リンクをクリックすると、ユーザー情報編集ページへ遷移します。
    ただし、変更できるのは、ユーザー名とパスワードのみとなります。
